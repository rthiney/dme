What is required to create a new request?
Estimated time to create a request is 2-3 minutes. To create a request, all the items below are required:
  1. ICD-10 code(s)
  2. Ordering Physician NPI and Fax Number
  3. HCPCS Code(s)
  4. Supporting Documentation (Example: H&P, Office Visit Note and Discharge Summary)
  5. Signed Physician's Order
If you do not have this information, please obtain and return to the portal to create the request. Requests created through the portal with complete information will be submitted and qualify for decision.
